# Plasma
오픈소스 PHP 프레임워크

## 설치
### 윈도에 [APMSETUP](http://www.apmsetup.com/)을 설치했을 경우
```
cd C:\APM_Setup\htdocs
git clone https://github.com/Hanul/Plasma.git
```

## Plasma 불러오기
```php
include 'Plasma/include.php';
```

## 업데이트
### 윈도에 [APMSETUP](http://www.apmsetup.com/)을 설치했을 경우
```
cd C:\APM_Setup\htdocs
cd Plasma
git pull
```
